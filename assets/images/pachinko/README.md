# Pachinko Pixel Art Assets

Please provide the following pixel art sprites:

## Required Assets

1. **peg_normal.png** (32×32 px)
   - Orange peg in normal state
   - Transparent background (.png with alpha)

2. **peg_hit.png** (32×32 px)
   - Golden/yellow peg when hit
   - Can include glow effect
   - Transparent background

3. **treat.png** (32×32 px)
   - Dog bone shape
   - Tan/brown colors
   - Transparent background

4. **puppy_normal.png** (64×64 or 128×128 px)
   - Puppy face with normal expression
   - Round eyes
   - Transparent background

5. **puppy_happy.png** (64×64 or 128×128 px)
   - Puppy face with happy/smiling expression
   - Transparent background

## Notes
- All assets should be PNG format with transparency
- Use pixel art style (no anti-aliasing, hard edges)
- Keep consistent pixel density across all sprites
